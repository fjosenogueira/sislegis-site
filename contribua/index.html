Seja um desenvolvedor do SISLEGIS!

O SISLEGIS é software livre. Qualquer um pode participar contribuindo de alguma forma! =) Para isso, basta ter uma conta no GitHub, fazer um fork de um dos projetos, verificar as issues que temos abertas para eles, codificar e submeter um pull request.

O SISLEGIS está dividido, neste momento, nos seguintes projetos:

  1. sislegis-site (este site). Para atuar nesse projeto, primeiro leia a página "Como contribuir na construção deste site".

    1. Issues abertas

  2. sislegis-prototipo. Protótipo de telas da aplicação, desenvolvido pelo arquiteto da informação da equipe.

  3. sislegis-dotfiles. Conjunto de arquivos de configuração (e scripts) utilizados em ambientes do projeto (desenvolvimento, homologação, integração, produção).

  4. sislegis-ambiente. Conjunto de scripts utilizados, especificamente, para a montagem de ambientes do projeto.

  5. sislegis-ambiente-fedora. Contjunto de scripts utilizados, especificamente, para a montagem de ambientes no Fedora.

  6. sislegis-app (a aplicação executada na infraestrutura do MJ). Para atuar nesse projeto, primeiro leia a página "Arquitetura da aplicação".
